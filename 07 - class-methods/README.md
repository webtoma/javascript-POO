# Exercices JavaScript : Méthodes de Classe

Ce projet propose six exercices pour vous aider à comprendre et à pratiquer l'utilisation de méthodes de classe en JavaScript. Les exercices portent sur la création de méthodes de classe statiques, l'appel de méthodes, et l'utilisation de méthodes pour manipuler des objets.

## Exercice 1 : Méthode de Classe Statique

**Objectif** : Créez une méthode de classe statique en JavaScript.

1. Ouvrez le fichier `exercice1.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `MathUtil` avec une méthode de classe statique `calculerMoyenne` qui prend un tableau de nombres en tant que paramètre et renvoie la moyenne de ces nombres.

## Exercice 2 : Appel de Méthodes de Classe

**Objectif** : Apprenez à appeler des méthodes de classe en JavaScript.

1. Ouvrez le fichier `exercice2.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Utilisez la classe `MathUtil` du premier exercice pour calculer la moyenne d'un tableau de nombres.

## Exercice 3 : Méthodes pour Manipuler des Objets

**Objectif** : Utilisez des méthodes de classe pour manipuler des objets.

1. Ouvrez le fichier `exercice3.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Personne` avec une méthode de classe `modifierAge` qui prend un objet `personne` et une nouvelle valeur d'âge, et met à jour l'âge de la personne.

## Exercice 4 : Méthode d'Instance

**Objectif** : Créez une méthode d'instance pour une classe.

1. Ouvrez le fichier `exercice4.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Modifiez la classe `Personne` du troisième exercice pour ajouter une méthode d'instance `afficherDetails` qui affiche les détails de la personne.

## Exercice 5 : Manipulation d'Objets

**Objectif** : Utilisez des méthodes de classe pour manipuler des objets.

1. Ouvrez le fichier `exercice5.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Compteur` avec des méthodes de classe pour incrémenter et décrémenter une valeur de compteur.

## Exercice 6 : Méthodes de Classe Avancées

**Objectif** : Pratiquez l'utilisation de méthodes de classe avancées.

1. Ouvrez le fichier `exercice6.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Forme` avec des méthodes de classe pour calculer l'aire et le périmètre de formes géométriques telles que le carré, le cercle et le triangle.

Ces exercices vous aideront à comprendre et à pratiquer l'utilisation de méthodes de classe en JavaScript pour manipuler des objets et effectuer des opérations mathématiques. Amusez-vous bien !
