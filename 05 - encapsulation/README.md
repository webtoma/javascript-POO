# Exercices JavaScript : Encapsulation

Ce projet propose six exercices pour vous aider à comprendre et à pratiquer l'encapsulation en JavaScript. Les exercices portent sur la création de classes avec des propriétés privées, l'utilisation de méthodes d'accès et la démonstration du concept d'encapsulation.

## Exercice 1 : Création de Classes avec Propriétés Privées

**Objectif** : Créez des classes avec des propriétés privées en JavaScript.

1. Ouvrez le fichier `exercice1.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Personne` avec une propriété privée `_nom` et une propriété `_age`.

4. Ajoutez des méthodes d'accès pour obtenir et définir les valeurs des propriétés privées.

## Exercice 2 : Utilisation de Méthodes d'Accès

**Objectif** : Utilisez des méthodes d'accès pour interagir avec des propriétés privées.

1. Ouvrez le fichier `exercice2.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez des instances de la classe `Personne` de l'exercice précédent.

4. Utilisez les méthodes d'accès pour obtenir et définir les valeurs des propriétés privées.

## Exercice 3 : Encapsulation en Pratique

**Objectif** : Mettez en pratique l'encapsulation en créant une classe avec des méthodes privées.

1. Ouvrez le fichier `exercice3.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `CompteBancaire` avec une propriété privée `_solde` et des méthodes pour effectuer des opérations bancaires.

4. Utilisez l'encapsulation pour garantir que les opérations bancaires sont effectuées correctement.

## Exercice 4 : Utilisation de l'Encapsulation

**Objectif** : Utilisez l'encapsulation pour sécuriser l'accès aux données.

1. Ouvrez le fichier `exercice4.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez des instances de la classe `CompteBancaire` de l'exercice précédent.

4. Effectuez des opérations bancaires en utilisant les méthodes de la classe, en veillant à respecter les règles d'encapsulation.

## Exercice 5 : Encapsulation en JavaScript Natif

**Objectif** : Utilisez l'encapsulation en JavaScript natif.

1. Ouvrez le fichier `exercice5.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet JavaScript avec des propriétés privées et des méthodes publiques.

4. Utilisez l'encapsulation pour protéger les propriétés privées.

## Exercice 6 : Création de Classes Encapsulées

**Objectif** : Créez des classes encapsulées avec des méthodes privées.

1. Ouvrez le fichier `exercice6.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Machine` avec une propriété privée `_etat` et des méthodes pour effectuer des opérations spécifiques.

4. Utilisez l'encapsulation pour garantir que les opérations sont effectuées correctement et en toute sécurité.

Ces exercices vous aideront à comprendre et à pratiquer le concept d'encapsulation en JavaScript. Amusez-vous bien !
