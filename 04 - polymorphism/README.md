# Exercices JavaScript : Polymorphisme

Ce projet propose six exercices pour vous aider à comprendre et à pratiquer le polymorphisme en JavaScript. Les exercices portent sur la création de classes, la mise en œuvre de méthodes polymorphiques et la démonstration du concept de polymorphisme.

## Exercice 1 : Création de Classes

**Objectif** : Créez des classes en JavaScript.

1. Ouvrez le fichier `exercice1.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Animal` avec une méthode `crier`.

4. Créez deux sous-classes, par exemple `Chien` et `Chat`, qui héritent de `Animal`.

5. Dans chaque sous-classe, redéfinissez la méthode `crier` pour qu'elle produise un cri différent.

## Exercice 2 : Utilisation du Polymorphisme

**Objectif** : Utilisez le polymorphisme pour appeler des méthodes de différentes sous-classes.

1. Ouvrez le fichier `exercice2.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez des instances de `Chien` et `Chat` de l'exercice précédent.

4. Appelez la méthode `crier` sur chaque instance et observez le comportement polymorphique.

## Exercice 3 : Polymorphisme et Interfaces

**Objectif** : Mettez en œuvre le polymorphisme en utilisant des interfaces.

1. Ouvrez le fichier `exercice3.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une interface `Véhicule` avec une méthode `démarrer`.

4. Créez deux classes, par exemple `Voiture` et `Moto`, qui implémentent l'interface `Véhicule`.

5. Appelez la méthode `démarrer` sur chaque instance et observez le polymorphisme en action.

## Exercice 4 : Utilisation du Polymorphisme

**Objectif** : Utilisez le polymorphisme pour manipuler des objets de manière générique.

1. Ouvrez le fichier `exercice4.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Forme` avec une méthode `aire`.

4. Créez plusieurs sous-classes telles que `Cercle`, `Rectangle` et `Triangle`, qui redéfinissent la méthode `aire`.

5. Utilisez un tableau pour stocker des objets de différentes sous-classes et appelez la méthode `aire` de manière polymorphique.

## Exercice 5 : Polymorphisme en JavaScript Natif

**Objectif** : Utilisez le polymorphisme en JavaScript natif.

1. Ouvrez le fichier `exercice5.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un tableau contenant des objets de différentes classes.

4. Utilisez la boucle `for...of` pour parcourir le tableau et appelez une méthode polymorphique sur chaque objet.

## Exercice 6 : Création de Classes Polymorphiques

**Objectif** : Créez des classes qui peuvent être utilisées polymorphiquement.

1. Ouvrez le fichier `exercice6.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Shape` avec une méthode `draw`.

4. Créez plusieurs sous-classes telles que `Circle`, `Rectangle` et `Triangle`, qui redéfinissent la méthode `draw`.

5. Utilisez ces classes pour créer des formes et appelez la méthode `draw` de manière polymorphique.

Ces exercices vous aideront à comprendre et à pratiquer le concept de polymorphisme en JavaScript. Amusez-vous bien !
