# Exercices JavaScript : Manipulation d'Objets

Ce projet propose six exercices simples pour vous aider à pratiquer la manipulation d'objets en JavaScript. Les exercices portent sur la création d'objets, la modification de propriétés et l'accès aux valeurs.

## Exercice 1 : Création d'Objets

**Objectif** : Créez un objet JavaScript simple.

1. Ouvrez le fichier `exercice1.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet nommé `personne` avec deux propriétés : `nom` et `age`.

4. Affectez des valeurs à ces propriétés.

5. Utilisez `console.log` pour afficher l'objet dans la console.

## Exercice 2 : Modification de Propriétés

**Objectif** : Modifiez les propriétés d'un objet.

1. Ouvrez le fichier `exercice2.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `produit` avec des propriétés telles que `nom`, `prix` et `description`.

4. Modifiez les valeurs de certaines de ces propriétés.

5. Utilisez `console.log` pour afficher l'objet mis à jour dans la console.

## Exercice 3 : Accès aux Propriétés

**Objectif** : Accédez aux propriétés d'un objet.

1. Ouvrez le fichier `exercice3.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `étudiant` avec des propriétés telles que `nom`, `âge` et `note`.

4. Utilisez `console.log` pour afficher le nom et l'âge de l'étudiant dans la console.

## Exercice 4 : Ajout de Propriétés

**Objectif** : Ajoutez de nouvelles propriétés à un objet existant.

1. Ouvrez le fichier `exercice4.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `voiture` avec des propriétés telles que `marque` et `année`.

4. Ajoutez une nouvelle propriété `couleur` à l'objet.

5. Utilisez `console.log` pour afficher l'objet mis à jour dans la console.

## Exercice 5 : Suppression de Propriétés

**Objectif** : Supprimez des propriétés d'un objet.

1. Ouvrez le fichier `exercice5.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `ordinateur` avec des propriétés telles que `marque`, `processeur` et `ram`.

4. Supprimez la propriété `ram` de l'objet.

5. Utilisez `console.log` pour afficher l'objet mis à jour dans la console.

## Exercice 6 : Création de Liste d'Objets

**Objectif** : Créez une liste d'objets et accédez à leurs propriétés.

1. Ouvrez le fichier `exercice6.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une liste d'objets représentant des livres. Chaque objet devrait avoir des propriétés telles que `titre` et `auteur`.

4. Utilisez une boucle pour parcourir la liste d'objets et afficher le titre et l'auteur de chaque livre dans la console.

Ces exercices vous aideront à développer vos compétences en manipulation d'objets en JavaScript. Amusez-vous bien !
