# Exercices JavaScript : Modélisation Orientée Objet

Ce projet propose six exercices pour vous aider à comprendre et à pratiquer la modélisation orientée objet en JavaScript. Les exercices portent sur la création de classes, la définition de propriétés et de méthodes, ainsi que l'instanciation d'objets.

## Exercice 1 : Création de Classes

**Objectif** : Créez une classe de base pour modéliser un concept.

1. Ouvrez le fichier `exercice1.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Personne` avec les propriétés `nom` et `age`.

4. Instanciez un objet à partir de la classe `Personne` et affichez ses propriétés dans la console.

## Exercice 2 : Méthodes de Classe

**Objectif** : Ajoutez des méthodes à une classe pour effectuer des actions.

1. Ouvrez le fichier `exercice2.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Utilisez la classe `Personne` du premier exercice et ajoutez une méthode `sePresenter` qui affiche une introduction de la personne.

4. Appelez la méthode `sePresenter` sur un objet `Personne` et affichez le résultat dans la console.

## Exercice 3 : Héritage

**Objectif** : Créez une classe enfant qui hérite d'une classe parente.

1. Ouvrez le fichier `exercice3.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe parente `Animal` avec une méthode `manger`.

4. Créez une classe enfant `Chien` qui hérite de `Animal` et ajoutez une méthode `aboie`.

## Exercice 4 : Méthodes de Classe Avancées

**Objectif** : Pratiquez la création de méthodes de classe avancées.

1. Ouvrez le fichier `exercice4.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Utilisez la classe `Chien` du troisième exercice et ajoutez une méthode `manger` qui prend en compte la nourriture spécifique d'un chien.

4. Créez une instance de `Chien` et appelez la méthode `manger` avec un exemple de nourriture.

## Exercice 5 : Composition d'Objets

**Objectif** : Pratiquez la composition d'objets en créant une classe qui contient d'autres objets.

1. Ouvrez le fichier `exercice5.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Maison` qui contient plusieurs instances de la classe `Piece`.

4. Chaque instance de `Piece` doit avoir un nom (par exemple, "salon", "chambre", "cuisine").

5. Créez une instance de `Maison` et affichez ses pièces dans la console.

## Exercice 6 : Méthodes de Chaine de Prototypes

**Objectif** : Pratiquez l'ajout de méthodes à la chaîne de prototypes d'une classe.

1. Ouvrez le fichier `exercice6.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Personne` avec les propriétés `nom` et `age`.

4. Ajoutez une méthode `presentation` à la chaîne de prototypes de `Personne` qui retourne une chaîne de caractères d'introduction.

5. Instanciez un objet `Personne` et appelez la méthode `presentation`.

Ces exercices vous aideront à développer vos compétences en modélisation orientée objet en JavaScript. Essayez d'abord les exercices simples, puis passez aux exercices plus avancés pour approfondir votre compréhension. Amusez-vous bien !
