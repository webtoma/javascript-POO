# Exercices JavaScript : Méthodes d'Objets

Ce projet propose six exercices pour vous aider à pratiquer l'utilisation des méthodes des objets en JavaScript. Les exercices portent sur la création de méthodes, l'appel de méthodes et l'utilisation des méthodes intégrées.

## Exercice 1 : Création de Méthodes

**Objectif** : Créez une méthode pour un objet.

1. Ouvrez le fichier `exercice1.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `calculatrice` avec deux propriétés numériques.

4. Ajoutez une méthode `addition` à l'objet pour effectuer une addition.

5. Utilisez la méthode pour afficher le résultat de l'addition dans la console.

## Exercice 2 : Appel de Méthodes

**Objectif** : Appelez des méthodes sur un objet.

1. Ouvrez le fichier `exercice2.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `restaurant` avec une méthode `commanderPlat` qui prend un plat en argument.

4. Appelez la méthode pour commander un plat et affichez un message dans la console.

## Exercice 3 : Méthodes Intégrées

**Objectif** : Utilisez des méthodes intégrées dans JavaScript.

1. Ouvrez le fichier `exercice3.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une chaîne de caractères et utilisez la méthode `toUpperCase` pour la convertir en majuscules.

4. Utilisez `console.log` pour afficher la chaîne en majuscules dans la console.

## Exercice 4 : Création de Méthodes d'Objets

**Objectif** : Créez des méthodes pour un objet.

1. Ouvrez le fichier `exercice4.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `compteBancaire` avec des propriétés telles que `solde` et `déposer`.

4. Ajoutez des méthodes pour déposer de l'argent et retirer de l'argent du compte.

5. Utilisez les méthodes pour effectuer des opérations sur le compte et afficher le solde dans la console.

## Exercice 5 : Utilisation de Méthodes Intégrées

**Objectif** : Utilisez des méthodes intégrées pour manipuler un tableau.

1. Ouvrez le fichier `exercice5.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un tableau de noms.

4. Utilisez la méthode `push` pour ajouter un nouveau nom au tableau.

5. Utilisez `console.log` pour afficher le tableau mis à jour dans la console.

## Exercice 6 : Création de Méthodes d'Objets

**Objectif** : Créez des méthodes pour un objet.

1. Ouvrez le fichier `exercice6.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez un objet `listeDesCourses` avec des propriétés telles que `articles` et `ajouterArticle`.

4. Ajoutez des méthodes pour ajouter et supprimer des articles de la liste.

5. Utilisez les méthodes pour gérer la liste des courses et affichez-la dans la console.

Ces exercices vous aideront à développer vos compétences en utilisant les méthodes des objets en JavaScript. Profitez-en !
