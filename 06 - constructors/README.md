# Exercices JavaScript : Constructeurs

Ce projet propose six exercices pour vous aider à comprendre et à pratiquer l'utilisation de constructeurs en JavaScript. Les exercices portent sur la création de classes, l'initialisation d'objets à l'aide de constructeurs, et la création d'instances avec des propriétés et des méthodes.

## Exercice 1 : Création de Classe avec Constructeur

**Objectif** : Créez une classe avec un constructeur en JavaScript.

1. Ouvrez le fichier `exercice1.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Personne` avec un constructeur qui prend les paramètres `nom` et `age`.

4. Utilisez le constructeur pour créer une instance de la classe `Personne`.

## Exercice 2 : Initialisation d'Objets avec Constructeur

**Objectif** : Initialisez des objets en utilisant un constructeur.

1. Ouvrez le fichier `exercice2.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Produit` avec un constructeur qui initialise les propriétés `nom`, `prix` et `stock`.

4. Utilisez le constructeur pour créer plusieurs instances de la classe `Produit`.

## Exercice 3 : Méthodes dans les Constructeurs

**Objectif** : Ajoutez des méthodes aux objets créés à partir de constructeurs.

1. Ouvrez le fichier `exercice3.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Voiture` avec un constructeur qui initialise la propriété `marque`.

4. Ajoutez une méthode `demarrer` à la classe `Voiture` qui affiche "La voiture démarre !" lorsque vous l'appelez.

## Exercice 4 : Constructeurs Avancés

**Objectif** : Utilisez des constructeurs avancés pour créer des objets.

1. Ouvrez le fichier `exercice4.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Chien` avec un constructeur qui initialise les propriétés `nom`, `race` et `age`.

4. Ajoutez des méthodes à la classe `Chien` pour afficher des informations sur le chien, telles que "Nom : [nom], Race : [race], Age : [age]".

## Exercice 5 : Héritage avec Constructeurs

**Objectif** : Utilisez l'héritage avec des constructeurs pour créer des sous-classes.

1. Ouvrez le fichier `exercice5.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe de base `Animal` avec un constructeur qui initialise la propriété `nom`.

4. Créez une sous-classe `Chat` qui étend la classe `Animal` et ajoute une propriété `couleur`.

## Exercice 6 : Constructeurs en Pratique

**Objectif** : Mettez en pratique l'utilisation des constructeurs pour créer des objets complexes.

1. Ouvrez le fichier `exercice6.js` dans votre éditeur de code.

2. Vous trouverez un commentaire indiquant où écrire votre code.

3. Créez une classe `Produit` avec un constructeur qui initialise les propriétés `nom`, `prix` et `stock`.

4. Créez une classe `Panier` qui utilise des constructeurs pour créer des instances de produits et les ajouter au panier.

Ces exercices vous aideront à comprendre et à pratiquer l'utilisation des constructeurs en JavaScript pour créer des objets et des classes. Amusez-vous bien !
