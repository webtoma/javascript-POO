# Exercices JavaScript : Programmation Orientée Objet (POO)

Ce projet propose une série d'exercices visant à vous aider à comprendre les principes de la programmation orientée objet (POO) en JavaScript. Vous travaillerez sur la création de classes, la définition de propriétés et de méthodes, l'héritage, le polymorphisme et bien plus encore.

## Thèmes de la Programmation Orientée Objet (POO) en JavaScript

Ce projet propose une série d'exercices couvrant divers thèmes de la programmation orientée objet (POO) en JavaScript. Chaque thème vous aidera à développer vos compétences en matière de POO. Voici la liste des 10 thèmes inclus :

1. Objets en JavaScript
2. Méthodes
3. Héritage
4. Polymorphisme
5. Encapsulation
6. Constructeurs
7. Méthodes de Classe
8. Getters et Setters
9. Héritage Avancé
10. Modélisation Orientée Objet



## Comment Utiliser Ce Projet

1. Clonez ce dépôt sur votre machine locale en utilisant la commande suivante :

2. Naviguez dans le répertoire du projet

3. Ouvrez les fichiers README.md de chaque dossier pour réaliser les exercices dans votre éditeur de code préféré.

4. Testez vos solutions en ouvrant le fichier index.html dans un navigateur web. Les résultats de chaque exercice s'afficheront dans la console du navigateur. Vous pouvez aussi utiliser le site JSBin pour tester votre code.

5. Continuez à travailler sur les exercices, modifiez le code, expérimentez et apprenez !
